# Keyboard instructions:

Moving:<br>
`W` - forwards

`S` - backwards

`A` - left

`D` - right
   
<br>
Ambient manipulation:
<br>

`F` - turn on/off flashlight

`B` - turn on/off bloom

`Q` - lower the exposure

`E` - raise the exposure
